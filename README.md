# GHAS-Laboratorio.
Laboratorio de pruebas para GitHub Advanced Security (GHAS), enfocado en la detección de vulnerabilidades mediante CodeQL, Dependabot y Secret Scanning.

## Objetivos

- Identificar vulnerabilidades de seguridad en el código fuente.
- Analizar resultados de CodeQL.
- Detectar secretos expuestos mediante Secret Scanning.
- Revisar dependencias vulnerables con Dependabot.
- Aplicar correcciones y validar las mejoras de seguridad.

## Herramientas utilizadas

- GitHub Advanced Security
- CodeQL
- Dependabot
- Secret Scanning
- Python

## Vulnerabilidades a evaluar

- SQL Injection
- Command Injection
- Uso inseguro de eval()
- Credenciales hardcodeadas
- Secretos expuestos
- Dependencias vulnerables
